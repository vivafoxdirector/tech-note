# AIX

## 디렉토리 용량 확인
```shell
$> du -sm /opt
$> du -sk /opt
```
※k=1kbyte m=1Mbyte

## 현재 디렉토리 이하의 용량 확인
```shell
$> du -m | egrep -v /.+/
```
