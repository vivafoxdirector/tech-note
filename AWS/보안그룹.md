## SG 규칙 수정
Amazon EC2 API 또는 명령줄 도구를 사용하여 기존 규칙의 프로토콜, 포트 범위, 소스 및 대상 등을 업데이트하려 할 때 규칙을 수정할 수 없습니다. 대신 기존 규칙을 삭제하고 새 규칙을 추가해야 합니다.
