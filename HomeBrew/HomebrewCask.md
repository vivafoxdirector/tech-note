# Homebrew Cask

## 개요
Homebrew Cask는 기존 Homebrew 확장판이다.

## 사용방법
```
brew cask <command> [<options>] [<token>]
```

## cask 설치된 목록 확인
```
brew cask list
```

## cask 검색
```
brew cask search "google"
```

## cask 정보 보기
```
brew cask info google-chrome
```

## cask 설치
```
brew cask install <tokens>
```

## cask 제거
```
brew cask uninstall <tokens>
```

## cask 설치된것 전체 업데이트
```
brew cask upgrade
```

## cask 설치된것 특정 업데이트
```
brew cask upgrade <tokens>
```
